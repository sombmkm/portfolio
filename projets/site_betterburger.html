Site web - Better Burger

Pour ce projet, nous avions la tâche de réaliser un site web correspondant à une entreprise de food trucks que nous avons inventé, le mien étant une marque basée sur les burgers, dénomée Better Burger.

Ce site web a été entièrement réalisé par moi-même, avec pour majeure partie l'utilisation d'HTML et de CSS.
L'utilisation de Javascript est également présente pour les pages "à propos" et "horaires" puisqu'elles réagissent à l'interaction de l'utilisateur (apparition de la composition des burgers quand on clique sur le burger correspondant, affichage de la position sur la Google Maps en fonction du jour sur la page des horaires).
